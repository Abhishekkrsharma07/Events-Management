## 📌 Overview

The **Student Event Management System** is a web application designed to streamline how students discover, host, and participate in college events. By organizing events into clear categories and listings, the platform reduces unnecessary browsing effort and keeps students informed about the latest happenings on campus.

> 📉 Improved user navigation by organizing event categories and listings, reducing browsing effort by **30%**.

---

## ✨ Features

- 🗓️ **Event Hosting** — Students and organizers can create and publish events with full details
- 🙋 **Event Participation** — Easy registration and participation flow for attendees
- 🔔 **Real-Time Updates** — Stay up to date with the latest changes to events
- 🗂️ **Event Categories** — Browse events filtered by type or category for faster discovery
- 📋 **Event Listings** — Clean, organized listings with all event info at a glance
- 📱 **Responsive Design** — Fully optimized for desktop and mobile screens

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Abhishekkrsharma07/Events-Management.git
   cd Events-Management
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000) to view the app.
