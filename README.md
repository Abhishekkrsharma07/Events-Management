Event Management System
Introduction
The Event Management System is a full-stack web application designed to simplify the process of managing and booking events. It allows users to browse events, view details, and book tickets, while administrators can create and manage events efficiently.
Features
User Features
- Browse available events
- View event details
- Book tickets/events
- View booking history
Admin Features
- Add new events
- Update event details
- Delete events
- Manage bookings
Technology Stack
Frontend: React.js (Vite), Tailwind CSS
Backend: Node.js, Express.js
Database: MongoDB
Other Tools: JWT Authentication, ImageKit

Installation and Setup
1. Clone the repository
2. Navigate to backend and install dependencies
3. Create a .env file and configure variables
4. Run backend server
5. Navigate to frontend and run the development server
